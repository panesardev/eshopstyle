# NGX Starter

## A minimal Angular project starter template

Key features

- 🆕 Uses Angular 17.2
- 🔥 Firebase as backend database
- ✅ Tailwind as css utility library
- 📏 Minimalistic project structure

Npm scripts

```
// start ssr application
$ yarn start
```

```
// development server
$ yarn dev 
```

```
// build application
$ yarn build
```
